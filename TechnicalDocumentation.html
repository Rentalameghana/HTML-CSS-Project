Introduction

JavaScript is a versatile language used in front-end and back-end development.

It runs in the browser and on servers with Node.js.

console.log('Hello World');
  • Dynamic typing
  • First-class functions
  • Prototype-based
Variables

Variables store data values and are declared using var, let, or const.

Use let or const instead of var for better scope control.

let x = 5; const y = 10;
  • let is block-scoped
  • const is also block-scoped and immutable
  • var is function-scoped
Functions

Functions are reusable blocks of code that perform a task.

They can be declared or assigned to variables.

function greet(name) { return 'Hello ' + name; } const add = (a, b) => a + b;
  • Function declarations are hoisted
  • Arrow functions are more concise
  • Functions can be passed as arguments
Loops

Loops help you run the same code multiple times.

Common loop types are for, while, and do...while.

for (let i = 0; i < 5; i++) { console.log(i); }
  • Use break to exit a loop early
  • Use continue to skip an iteration
  • Be careful with infinite loops
Arrays

Arrays hold a list of values in JavaScript.

They are zero-indexed and dynamic.

let fruits = ['apple', 'banana', 'cherry'];
  • Access items using index: fruits[0]
  • Use push() to add items
  • Use pop() to remove items
